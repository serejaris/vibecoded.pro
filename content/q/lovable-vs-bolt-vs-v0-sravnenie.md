## Мнения из сообщества

### @lovable_user

> Lovable — единственный из трёх с визуальным редактором. Кликаешь на элемент, редактируешь напрямую. Не нужно описывать в чате «сдвинь кнопку правее» — просто двигаешь мышкой.

Визуальный контроль — главное преимущество Lovable перед конкурентами.

### @bolt_user

> Bolt работает прямо в браузере через WebContainer. Ничего не устанавливать — описываешь задачу, видишь как создаётся код в реальном времени, получаешь рабочее приложение. Поддерживает любой стек, не только React.

Bolt — самый быстрый старт, буквально ноль установок.

### @v0_user

> v0 делает красивый UI на shadcn/ui + Tailwind. Три минуты от промпта до публичной ссылки на Vercel. Кнопка «Починить» исправляет ошибки без промпта. Для компонентов — лучший из трёх.

v0 — специализирован на UI, не full-stack платформа.

### @mvp_founder

> Паттерн: Lovable создаёт MVP за вечер, потом синхронизируешь с GitHub и дорабатываешь в Cursor. Lovable как быстрый старт, Cursor для контроля. Работает отлично.

GitHub sync у Lovable — ключевая фича для серьёзных проектов.

### @free_tier_user

> Все три дают Free tier. v0 — 7 сообщений/день, хватает попробовать. Рабочий план у всех ~$20/мес. Начинай с бесплатного, переходи на платный когда нужно масштабировать.

### @after_builder

> После браузерного билдера обычно переходишь в Claude Code для архитектурных задач. Билдеры отлично делают MVP, для продакшена — нужна нормальная IDE.

Браузерные билдеры → IDE: стандартный путь роста.

## Итог

| Критерий | Lovable | Bolt | v0 |
|----------|---------|------|-----|
| Тип | Full-stack | Full-stack | UI генератор |
| Free tier | Да | Да | 7 сообщений/день |
| Рабочий план | $20/мес | $20/мес | $20/мес |
| Визуальный редактор | Да | Нет | Нет |
| Стек | React + Supabase | Любой | React + Tailwind |
| GitHub sync | Да | Экспорт | Нет |
| Для кого | MVP с визуальным контролем | Быстрый прототип | UI-компоненты |

**Выбирай Lovable** — если нужен визуальный редактор и готовый бэкенд (Supabase).

**Выбирай Bolt** — если нужно максимально быстро без установок, любой стек.

**Выбирай v0** — если нужны красивые React-компоненты или быстрый лендинг.
